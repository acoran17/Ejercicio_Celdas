Game

Objetivo

Si eliminas todas las celdas, ganas. Si las celdas llegan al final del recuadro, pierdes.

Reglas

Se crean un numero de celdas de colores aleatorios de inicio.
Cada poco tiempo aparecera una nueva celda de un color aleatorio a la derecha.
Pulsa una celda y si hay mas de 3 celdas del mismo color juntas, incluida la pulsada, se borraran.
Pulsa una celda teniendo pulsado Ctrl y se hara un intercambio con la celda de la derecha.
